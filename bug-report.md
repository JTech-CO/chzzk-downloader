# Chzzk Downloader v2.2.2 업데이트 안내

안녕하세요, 치지직 다운로더 개발자입니다.

이번 **v2.2.2**는 라이브 방송 시청 중 화면 우측 하단 다운로드 아이콘이 채팅창을 가리는 문제를 해결한 UI 개선 업데이트입니다.

---

## 1. 라이브 방송 화면에서 다운로드 아이콘 노출 문제 해결

**기존 문제**
다운로드 아이콘이 치지직의 모든 페이지에서 표시되어, 라이브 방송 시청 중에는 우측 하단 채팅창 또는 방송 화면 조작 영역을 가리는 불편이 있었습니다.

**해결**
URL에 `live`가 포함된 라이브 방송 페이지에서는 다운로드 아이콘과 패널을 표시하지 않도록 변경했습니다. 이제 다운로드 UI는 실제 다운로드 대상이 있는 **동영상(VOD)** 탭과 **클립** 탭에서만 나타납니다.

---

## 2. 지원 페이지 표시 기준 명확화

v2.2.2부터 확장 프로그램의 UI 표시 기준은 다음과 같습니다.

| 페이지 유형 | URL 포함 값 | 다운로드 아이콘 표시 |
|-------------|-------------|----------------------|
| 동영상(VOD) | `videos` | 표시 |
| 클립 | `clips` | 표시 |
| 라이브 방송 | `live` | 표시 안 함 |
| 그 외 페이지 | 기타 | 표시 안 함 |

치지직은 페이지 전환이 빠른 SPA 구조이므로, 동영상/클립/라이브 화면 사이를 이동해도 URL 변화를 감지해 아이콘 표시 상태를 자동으로 갱신합니다. 라이브 방송으로 이동하면 열려 있던 패널도 함께 닫힙니다.

---

## 업데이트 방법

기존에 설치된 확장 프로그램을 삭제한 뒤, 깃허브 또는 크롬 웹스토어에서 최신 **v2.2.2** 버전을 설치해 주세요. 동일 확장의 구버전이 있다면 먼저 제거 후 설치를 권장합니다.

제보해 주신 모든 분들께 진심으로 감사드립니다. 앞으로도 실제 사용 흐름에서 거슬리는 부분을 빠르게 줄여가는 치지직 다운로더가 되겠습니다.
