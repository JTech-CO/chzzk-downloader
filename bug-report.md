# Chzzk Downloader v2.2.4 업데이트 안내

안녕하세요, 치지직 다운로더 개발자입니다.

이번 **v2.2.4**는 장시간 VOD 다운로드 안정성과 확장 프로그램 보안 경계를 강화한 업데이트입니다. 특히 10~12시간처럼 매우 긴 방송 다시보기도 정상적인 디스크 스트리밍 경로에서는 계속 다운로드할 수 있도록 유지하면서, 비정상 응답으로 인한 메모리/디스크 폭주 가능성을 줄였습니다.

---

## 1. 장시간 VOD 다운로드 안전성 강화

v2.2.3에서 HLS/fMP4 호환성을 개선하면서 Range 요청 처리가 추가되었습니다. 일부 환경에서는 서버가 Range 요청을 무시하고 전체 파일을 응답할 수 있어, 매우 긴 VOD에서 브라우저와 OS에 과도한 부담이 생길 수 있었습니다.

v2.2.4에서는 다음 조건을 검증합니다.

- Range 요청은 반드시 `206 Partial Content` 응답일 때만 처리
- 예상 Range 크기보다 큰 응답은 중단
- 단일 세그먼트가 비정상적으로 큰 경우 중단
- 응답 본문을 읽는 중에도 크기 제한을 초과하면 중단

긴 영상이라는 이유만으로 중간에 취소하지는 않습니다. 정상적인 세그먼트와 OPFS 디스크 스트리밍 경로가 유지되는 경우에는 장시간 VOD도 계속 다운로드합니다.

---

## 2. 메시지 및 URL 검증 추가

백그라운드 다운로드 요청은 치지직 탭에서 온 메시지만 처리하도록 제한했습니다. 또한 다운로드 URL과 세그먼트 URL은 HTTPS 공개 주소이면서 Naver/Pstatic 계열 도메인인 경우만 허용합니다.

이를 통해 의도하지 않은 다운로드 요청이나 잘못된 URL이 백그라운드 다운로드 경로로 들어오는 위험을 줄였습니다.

- 로컬호스트, 사설망 IP, 비정상 Range 입력 차단
- offscreen 문서로 전달되는 임시 파일명 검증
- 확장 UI에서 사용하던 외부 폰트 CDN 호출 제거

---

## 3. 권한 및 배포 패키지 구성 고정

매니페스트의 호스트 권한을 `<all_urls>`에서 `chzzk.naver.com`, `*.naver.com`, `*.pstatic.net` 범위로 줄였습니다. 또한 배포용 zip에 테스트 파일이나 문서가 섞이지 않도록 whitelist 패키징 스크립트를 추가했습니다. 이제 확장 프로그램 실행에 필요한 파일만 zip에 포함됩니다.

---

## 업데이트 방법

기존에 설치된 확장 프로그램을 삭제한 뒤, 깃허브 또는 크롬 웹스토어에서 최신 **v2.2.4** 버전을 설치해 주세요. 동일 확장의 구버전이 있다면 먼저 제거 후 설치를 권장합니다.
