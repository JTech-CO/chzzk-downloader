# Chzzk Downloader v2.2.3 업데이트 안내

안녕하세요, 치지직 다운로더 개발자입니다.

이번 **v2.2.3**은 Mac에서 일부 다운로드 파일이 재생되지 않던 문제를 수정하고, 동영상/클립 목록이 16개 또는 24개까지만 보이던 제한을 개선한 업데이트입니다.

---

## 1. Mac에서 다운로드 파일이 재생되지 않던 문제 수정

**기존 문제**
일부 Mac 환경에서 다운로드된 영상 파일을 열면 `mdat`는 있지만 `moov`가 없다는 형태의 오류가 발생하며 재생되지 않는 사례가 있었습니다. 이 문제는 `inKey` 유무와 관계없이 HLS/fMP4 방식으로 저장되는 영상에서 발생할 수 있었습니다.

**원인**
HLS/fMP4 플레이리스트에서는 실제 영상 데이터 조각과 별도로 `#EXT-X-MAP` 초기화 세그먼트가 제공될 수 있습니다. 이 초기화 세그먼트에는 MP4 재생에 필요한 `ftyp`/`moov` 메타데이터가 들어갑니다. 기존 로직은 영상 데이터 조각만 병합해 `mdat`는 존재하지만 `moov`가 빠진 MP4가 만들어질 수 있었습니다.

**해결**
HLS 미디어 플레이리스트의 `#EXT-X-MAP`을 파싱해 초기화 세그먼트를 실제 영상 조각 앞에 함께 병합하도록 수정했습니다. `BYTERANGE`가 붙은 초기화 세그먼트도 처리하여, macOS QuickTime 계열 플레이어에서도 정상 재생 가능한 MP4 파일을 만들도록 개선했습니다.

---

## 2. 동영상/클립 목록이 중간에 잘리던 문제 개선

**기존 문제**
채널에 동영상이나 클립이 더 많아도 패널에는 최대 16개 또는 24개 정도까지만 표시되는 문제가 있었습니다. 첫 API 페이지만 조회하거나, 화면 DOM 스캔 결과가 API 목록을 덮어쓰면서 목록이 줄어드는 경우가 있었습니다.

**해결**
목록 API를 한 번만 호출하지 않고 여러 페이지를 순차적으로 조회하도록 변경했습니다. 또한 DOM 스캔 결과는 기존 API 목록을 덮어쓰지 않고 빠진 항목만 보강하도록 수정했습니다. 이제 더 많은 동영상과 클립이 있는 채널에서도 가능한 전체 목록을 패널에서 확인할 수 있습니다.

## 3. 최신순·과거순·인기순 정렬 추가

패널 상단에 정렬 버튼을 추가했습니다.

| 정렬 | 설명 |
|------|------|
| 최신순 | 기본값이며, 최근 업로드된 항목부터 표시합니다. |
| 과거순 | 오래된 항목부터 표시합니다. |
| 인기순 | 조회수가 높은 항목부터 표시합니다. |

정렬은 사용자의 브라우저 안에서 로컬로 처리되며, 별도의 서버로 이용 기록이나 목록 정보를 전송하지 않습니다.

제보해 주신 모든 분들께 진심으로 감사드립니다. 앞으로도 실제 사용 흐름에서 발견되는 호환성 문제와 불편을 빠르게 개선하겠습니다.
