Chzzk Downloader 개인정보 처리방침

최종 업데이트: 2026년 4월 16일

1. 개인정보 수집 및 이용

Chzzk Downloader는 사용자 개인을 식별할 수 있는 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

본 확장 프로그램은 오직 네이버 치지직(Chzzk)에서 제공되는 동영상(VOD) 및 클립의 다운로드 기능을 제공하는 목적으로만 동작합니다.

2. 권한(Permissions) 사용 목적

확장 프로그램이 원활히 동작하기 위해 구글 크롬 웹스토어 정책에 따라 다음과 같은 최소한의 권한을 요청합니다:

  • activeTab: 현재 활성화된 탭이 치지직(chzzk.naver.com) 사이트인지 확인하고, 화면 우측 하단에 다운로드 사용자 인터페이스(패널)를 주입하기 위해 사용됩니다.
  • downloads: 백그라운드 환경에서 다운로드된 동영상/클립 세그먼트를 병합하여 완성된 MP4 파일을 사용자의 로컬 디바이스에 저장하기 위해 사용됩니다.
  • 호스트 권한 (*.naver.com 등): 동영상 스트리밍 데이터를 가져오기 위한 API 호출 목적으로 필요합니다. 정상적인 화질 제공을 위해 로그인된 사용자의 세션을 통해 우회 없이 직접 다운로드되며, 이러한 데이터 통신은 사용자의 브라우저 내에서만 이루어지고 외부 서버로 어떠한 데이터도 전송되지 않습니다.

3. 제3자 데이터 제공

수집하는 데이터가 원천적으로 없으므로 제3자에게 데이터를 제공하거나 공유하는 과정이 존재하지 않습니다. 확장 프로그램의 모든 통신과 다운로드 처리 과정은 사용자와 네이버 서버 간 직접 이루어지며, 제3자 서버를 경유하지 않고 사용자의 로컬 브라우저 내에서 완료됩니다.

4. 개인정보 처리방침의 변경

본 확장 프로그램의 업데이트나 구글 크롬 웹스토어 정책 변경 등에 따라 이 개인정보 처리방침의 내용이 변경될 수 있습니다. 변경사항이 발생할 경우 버전 업데이트를 통해 안내되며, 본 페이지가 지속적으로 최신화될 것입니다.
